Installation instructions 

Pre-requirements

DJDisplay require wxpython 3.0. On Ubuntu this can be installed by:

sudo apt-get-get install python-wxgtk3.0

V0.1

This version only support local installation. Unpack the application in the desired folder in your home folder or other folder you have read and write privileges.

Note: this version require access to write a default settings file which must be located in an area where you as user have these privileges.

$~/ tar -zxvf DJDisplay-0.1.tar.gz

Make the main script executable

$~/ cd DJDisplay
